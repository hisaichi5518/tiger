# Tiger

Tiger is a mixin class to on/emit triggers (or hooks) that get called at some points you specify.

## Installation

Add this line to your application's Gemfile:

    gem 'tiger'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tiger
